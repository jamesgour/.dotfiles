Download Microsoft Keyboard Layout Creator (MSKLC) to edit the .klc file in this folder
Go to file > Load Existing Keyboard > Real Programmer Dvoark.klc
Save a new version if you want to make edits
Go to Project > Properties and input properties
Once done, Project > Validate Layout
Project > Test Keyboard Layout
Project > Build DLL & Setup Package
Optionally, File > Save As Image to print out a reminder
Move the directory with the name of the layout to your desktop
Click Setup.exe and follow instructions > should say keyboard was installed correctly
